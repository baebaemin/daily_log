## Today I Learned

### [git] 로컬 파일은 삭제하지 않고 캐시 제거하기

- 특정 폴더와 파일을 커밋 내역에 올라가지 않도록 `.gitignore`을 수정했으나, 수정 전 이미 파일이 git add 되어 staging area에 올라가 있었기 때문에 커밋으로 추적하게 되었다. 아래와 같은 명령어로 로컬 파일은 그대로 둔 채 캐시만 삭제하고, 커밋 내역을 수정했다.

```bash
# 캐시에서 제거 (파일은 삭제되지 않음)
git rm --cached GEMINI.md

# -r(--recursive)는 재귀적 작업으로 폴더 삭제시 추가하는 옵션
git rm -r --cached .agent/

# 다시 커밋
git commit --amend --no-edit
```
