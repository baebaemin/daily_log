## Today I Learned

### [Web] Base64

- Binary 데이터를 64개의 안전한 ASCII 문자로 변환하는 인코딩 방식
  - A-Z (26개)
  - a-z (26개)
  - 0-9 (10개)
  - `+`, `/` (2개)
  - `=` (패딩용)

- 이미지나 파일 같은 바이너리 데이터를 텍스트만 전송할 수 있는 포맷 (이메일, URL, JSON, JWT 토큰, API 인증 헤더 등) 전송해야할 때 Base64로 인코딩하여 사용

  ```html
  <!-- 이미지를 Base64로 인코딩하여 HTML에 직접 포함 -->
  <img src="data:image/png;base64,iVBORw0KGgoAAAANS..." />
  ```

- 참고 링크 : https://developer.mozilla.org/ko/docs/Glossary/Base64
