| [TIL] CSS 두 번째 줄 넘어갔을 때 부터 말줄임표(...)사용하고 싶을 때

```
.ellipsis {
  width: 200px;
  overflow: hidden;
  text-overflow: ellipsis;
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
}
```
