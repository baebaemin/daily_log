### [Marketing Tools] Braze

- 마케팅 자동화 도구 Customer Engagement Platform
  - 앱/웹 푸시 메시지 전송
  - 앱 사용 중 팝업, 배너 표시
  - 자동화된 이메일, SMS 발송
  - 사용자 행동 기반 그룹화
  - 사용자별 맞춤 메세지로 개인화된 마케팅

```tsx
// MarketingTools.tsx

const BRAZE_CONFIG: BrazeConfig = {
  baseUrl: "sdk.iad-05.braze.com", // 공식 SDK 공개 엔드포인트
  allowUserSuppliedJavascript: true,
  minimumIntervalBetweenTriggerActionsInSeconds: 5,
  inAppMessageZIndex: 12000,
};

const initBraze = (memIdx: number, name: string) => {
  braze.initialize(process.env.NEXT_PUBLIC_BRAZE_API_KEY!, BRAZE_CONFIG); // Braze 연결 설정
  braze.automaticallyShowInAppMessages();
  braze.changeUser(String(memIdx)); // 사용자 식별
  braze.getUser()?.setFirstName(name); // 사용자 이름 설정
  braze.openSession(); // 세션 시작 (정보 전송 완료)
  window.braze = braze; // 초기화된 braze 객체를 전역으로 저장
};
```

- Google Analytics: "누가 무엇을 했는지" 분석
- Braze: "누구에게 무엇을 보낼지" 마케팅 자동화
- Amplitude: 분석 중심
