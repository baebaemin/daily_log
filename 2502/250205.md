## Today I Learned

### [TIL-WORDS]

- [`stable sort`](https://en.wikipedia.org/wiki/Sorting_algorithm#Stability): 같은 값을 가진 원소들이 정렬 후에도 순서가 보존되는 것

  ```js
  [7(🔵), 5(🟡), 2(🟢), 5(🔴)]

  // stable sort
  [2(🟢), 5(🟡), 5(🔴), 7(🔵)]

  // unstable sort
  [2(🟢), 5(🔴), 5(🟡), 7(🔵)]
  ```

  - stable sort 예시

    - insertion sort (삽입 정렬)
    - bubble sort (버블 정렬)
    - merge sort (병합 정렬)
    - heap sort (힙 정렬)
    - quick sort (퀵 정렬)
    - selection sort (선택 정렬)

  - unstable sort 예시
    - counting sort (계수 정렬)
    - radix sort (기수 정렬)
    - bucket sort (버킷 정렬)

### [JavaScript] .sort()

- 정렬 순서는 문자열의 유니코드 코드 포인트를 따르며, stable sort가 아닐 수 있다.

```js
const months = ["March", "Jan", "Feb", "Dec"];
months.sort();
console.log(months);
// Expected output: Array ["Dec", "Feb", "Jan", "March"]

const array1 = [1, 30, 4, 21, 100000];
array1.sort();
console.log(array1);
// Expected output: Array [1, 100000, 21, 30, 4]
```

- 참고 링크 : https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/sort
