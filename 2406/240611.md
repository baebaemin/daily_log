### [CSS] width, height auto

- 240520에서 작성했던 onImgLoad보다 훨씬훨씬 간편한 방법이 있었다..

```javascript
// Next Image
<Image
  className={cx("content__img")}
  src={`${imgPath}`}
  // onLoad={onImgLoad}
  width={664}
  height={369}
  alt={"상품 미리보기"}
  priority
  unoptimized
/>
```

- width랑 height 특정 값으로 지정해두고 ratio를 유지하면서 지정한 너비와 높이 안에서 렌더링하려면

```scss
.content {
  &__img {
    width: auto;
    height: auto;
  }
}
```

- 이렇게만 해줘도 된다
