### [Web] Client/Server에 따른 API 엔드포인트 분리

```js
NEXT_PUBLIC_KEYSTONE_API = https://dev.api-service.saramin.co.kr
NEXT_PUBLIC_KEYSTONE_API_HOST = http://dev.api-service.saramin.host
```

- `co.kr` : 공개 도메인

  - 외부에서 접근 가능
  - HTTPS 프로토콜 사용
  - client side에서 브라우저를 통해 API를 호출할 때 사용

- `.host` : 내부 호스트명
  - HTTP 프로토콜 사용
  - 로컬 네트워크나 내부 DNS에서만 해석되는 도메인
  - server side에서 내부 네트워크를 통해 API를 호출할 때 사용 (SSR, API routes 등)
  - 인터넷을 거치지 않고 더 빠르고 안전하게 통신할 수 있다
