## Today I Learned

### [JavaScript] atob(), btoa()

- btoa
  - Binary to ASCII의 약자
  - 문자열을 Base64로 인코딩

  ```js
  const encoded = btoa("Hello World");
  console.log(encoded); // "SGVsbG8gV29ybGQ="
  ```

- atob
  - ASCII to Binary의 약자
  - Base64로 인코딩된 문자열을 디코딩

  ```js
  const decoded = atob("SGVsbG8gV29ybGQ=");
  console.log(decoded); // "Hello World"
  ```

- btoa와 atob는 브라우저와 Node.js(18+)에서 사용 가능
- btoa()는 ASCII 문자만 처리할 수 있어서, 한글 등 유니코드 문자를 인코딩하려면 추가 처리가 필요하다.

  ```js
  // 한글 인코딩 (브라우저)
  const encoded = btoa(encodeURIComponent("안녕하세요"));

  // 한글 디코딩 (브라우저)
  const decoded = decodeURIComponent(atob(encoded));
  ```
