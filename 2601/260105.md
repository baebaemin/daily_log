### [Article] 웹과 다른 세계: 이메일 템플릿 작업에서 겪은 시행착오

- 이메일 클라이언트는 서로 다른 내부 렌더링 엔진을 사용한다.

  - Outlook (windows): MS Word 렌더링 엔진
  - Gmail : CSS 필터링 + 자체 파서
  - iOS Mail : WebKit 기반
  - 네이버 / 카카오 메일 : 제한적인 WebView

- CSS를 웹 브라우저처럼 정확히 그려주지 못하지만 table 레이아웃은 안정적으로 지원
