### [CSS] :nth-of-type()

- `box_items` div 태그의 하위 태그들 중 span 태그의 두 번째 형제에만 특정 css효과를 주고 싶다면 `:nth-of-type()`을 활용하자
- index는 1부터 시작한다

```javaScript
<div className="box_items">
    <span>item 1</span>
    <span>item 2</span>
</div>
<div className="box_items">
    <span>item 3</span>
    <span>item 4</span>
</div>
```

```scss
// style.scss
.box_items {
    ...
    span {
        ...
        &:nth-of-type(2) {
           ...
        }
    }
}
```

- 참고링크 : https://developer.mozilla.org/en-US/docs/Web/CSS/:nth-of-type
