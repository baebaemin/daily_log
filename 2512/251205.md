### [Next.js] useRouter() 사용으로 뒤로가기시 이전 스크롤 위치 유지하기

- 기본적으로 Next.js는 새 route로 navigating할 때 페이지 상단으로 스크롤을 이동시킨다. router.push() 또는 router.replace() 사용시 `{ scroll: false }` 옵션을 전달함으로써 이전 스크롤을 유지시킬 수 있다.

```tsx
return (
  <Header
    depthCategory="sub-depth"
    title="공고 수정"
    leftBtnType={{
      value: {
        iconName: "back-arrow",
        onClick: () => router.push("/hiring/manage", { scroll: false }), // 화면 위로 스크롤 위치 업데이트 방지
      },
    }}
  />
);
```

- 관련 링크 : https://nextjs.org/docs/app/api-reference/functions/use-router#disabling-scroll-to-top
