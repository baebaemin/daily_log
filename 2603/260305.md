## Today I Learned

### [React] 리액트의 useEffectEvent 이해하기: 오래된 클로저 해결을 위한 완벽 가이드

useEffect를 사용하면서도 useEffect의 규칙, 작동 방식을 우회하고 싶은 상황 때문에 아래와 같이 구리구리 코드를 작성했었다.

- smell 1 : `// eslint-disable-next-line react-hooks/exhaustive-deps` 남용
  - useEffect 내부에서 어떤 값을(주로 상태나 api에서 받아오는 값) 사용하나, 해당 값이 변할 때마다 렌더링하고싶지 않을 때. 보통 아래와 같은 종류의 NOTE line을 같이 달아두어 팀원님들께 양해를... 구한다 ^-^...
    - `// NOTE: 렌더링 후 단 한 번만 실행시키고자 eslint를 disabled 처리`
    - `// NOTE: 불필요한 리렌더링 방지용으로 headerHeight, setHeaderHeight 제거`

- smell 2 : `useRef` 사용
  - useRef로 ref 객체를 생성해 current 안에 값을 담아두는 방식. useEffect 내부에서 current를 참조해 사용해도 값이 변하지 않다보니 의존성 배열에도 추가할 필요가 없었다. 하지만 보일러 플레이트 코드가 꽤 많고, DOM 측정값을 ref에 동기적으로 담아야 할 때에는 useLayoutEffect를 병행해서 써야했다. (resizeObserver 등...)

`useEffectEvent`를 사용하면 많은 부분이 해결된다는 걸 [이 아티클(번역)](https://handhand.tistory.com/entry/understanding-react-useeffectevent)을 통해 알게되어 정리 및 적용해보았다.

### smell 개선 : Effect와 Event를 분리 - 1

반응할 값과 읽고 수정할 값의 분리

[AS-IS] 칸반 보드의 카드들을 체크했을 때, 칸반 Column에 속한 카드들이면 checkedApplicants store값을 업데이트하는 로직이 있었는데 이때 useEffect의 의존성 배열에 값이 불필요하게 많았다.

```tsx
const checkedValues = useApplicantsStore((s) => s.checkedValues);
const setCheckedApplicants = useApplicantsStore((s) => s.setCheckedApplicants);

/**
 * 현재 전형과 체크한 후보자가 있는 전형이 같은지 여부
 ** NOTE: currentProcessSeq는 store에서 구독 중인 값이므로, 값이 변경될 때마다
 ** 컴포넌트가 리렌더링되어 isSameProcess도 항상 최신 상태로 유지되는 중
 */
const isSameProcess = currentProcessSeq === hiringProcessSeq;

/** 현재 전형과 체크한 후보자가 있는 전형이 다를 경우 checkedValues 배열 초기화 */
useEffect(() => {
  if (isSameProcess) {
    const applicants = getCheckedApplicants(checkedValues, applyList);
    setCheckedApplicants(applicants || EMPTY_ARRAY);
  }
}, [checkedValues, setCheckedApplicants, applyList, isSameProcess]);
```

<br/>[TO-BE] 체크할 때 바뀌는 checkedValue만 주목하고 데이터 동기화 기능은 useEffectEvent에 담았다.

```tsx
const checkedValues = useApplicantsStore((s) => s.checkedValues);
const setCheckedApplicants = useApplicantsStore((s) => s.setCheckedApplicants);

/** 현재 전형과 체크한 후보자가 있는 전형이 같은지 여부 */
const isSameProcess = currentProcessSeq === hiringProcessSeq;

/**
 * 현재 전형과 체크한 후보자가 있는 전형이 같을 경우 checkedValues 데이터 동기화
 ** NOTE: useEffectEvent로 만들었기 때문에 stable한 참조
 */
const syncCheckedApplicants = useEffectEvent(() => {
  if (isSameProcess) {
    const checkedApplicants = getCheckedApplicants(checkedValues, applyList);
    setCheckedApplicants(checkedApplicants || EMPTY_ARRAY);
  }
});

// 체크한 후보자 값 달라질 때 실행
useEffect(() => {
  syncCheckedApplicants();
}, [checkedValues]);
```

### 2. smell 개선 : Effect와 Event를 분리 - 2

리스트를 성공적으로 가져왔을 때 로그를 남기는 부분

[AS-IS] Effect 내부에서 쓰이는 applyList는 infinite query로 불러오는 값이라 계속 변할 수 있어 의존성 배열에 추가하지 않고 lint 무효화 주석이 필요했다.

```tsx
/** 해당 전형의 후보자 리스트 */
const applyList =
  data?.pages
    .flatMap((page) => page.applyList)
    .filter((item): item is ManageApplicant => !!item) ?? [];

useEffect(() => {
  if (isSuccess && applyList && applyList.length > 0) {
    getApplicantsLogging({
      recIdx,
      recruitList: applyList.map((data) => data.recruitapplyIdx),
    });
  }
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, [isSuccess, recIdx]);
```

<br/> [TO-BE] 로깅하는 부분만 useEffectEvent로 분리

```tsx
/** 해당 전형의 후보자 리스트 */
const applyList =
  data?.pages
    .flatMap((page) => page.applyList)
    .filter((item): item is ManageApplicant => !!item) ?? [];

/** 후보자 리스트 1명 이상 불러왔을 때 로깅 */
const logPageEntry = useEffectEvent(() => {
  if (isSuccess && applyList && applyList.length > 0) {
    getApplicantsLogging({
      recIdx,
      recruitList: applyList.map((data) => data.recruitapplyIdx),
    });
  }
});

useEffect(() => {
  logPageEntry();
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, [isSuccess]);
```

### 3. smell 개선 : 필요없는 useRef 제거

첫 렌더링 때 DOM을 모니터링할 observer를 만든 후 headerHeight 초기값을 계산

[AS-IS]

- useEffect 안에 너무 많은 로직이 들어있다.
  - [생성] : ResizeObserver 인스턴스 생성
  - [구독] : DOM(ref.current) 감시
  - [초기화] : 현재 시점의 headerHeight 계산하여 초기화
  - [정리] : 컴포넌트가 사라질 때 감시 중단하고 타이머 제거

```tsx
const ref = useRef<HTMLDivElement>(null);
const headerHeight = useHeaderHeightStore((s) => s.headerHeight);
const setHeaderHeight = useHeaderHeightStore((s) => s.setHeaderHeight);
const timeoutRef = useRef<ReturnType<typeof setTimeout> | null>(null);

useEffect(() => {
  if (!ref.current) return;

  // [생성]
  const resizeObserver = new ResizeObserver((header) => {
    // 이전 타이머 취소
    if (timeoutRef.current) clearTimeout(timeoutRef.current);

    // 디바운스를 통해 빈번한 업데이트 방지
    timeoutRef.current = setTimeout(() => {
      const observingHeight = header[0].contentRect.height;

      // 이전 값과 비교하여 실제로 변경되었을 때만 업데이트
      if (Math.abs(lastHeightRef.current - observingHeight) >= 1) {
        lastHeightRef.current = observingHeight;
        setHeaderHeight(observingHeight || 0);
      }
    }, 16); // ~60fps에 맞춘 디바운스
  });

  // [초기화]
  const initialHeight = Math.round(ref.current.offsetHeight);
  if (initialHeight !== headerHeight) {
    lastHeightRef.current = initialHeight;
    setHeaderHeight(initialHeight);
  }

  // [구독]
  resizeObserver.observe(ref?.current);

  // [정리]
  // eslint-disable-next-line consistent-return
  return () => {
    resizeObserver.disconnect();
    if (timeoutRef.current) {
      clearTimeout(timeoutRef.current);
    }
  };
  // 의존성 최적화를 위해 headerHeight, setHeaderHeight 제거
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, []);
```

[TO-BE]

- ResizeObserver의 재성성 없이 동작
- 불필요한 lastHeightRef.current 대신 스토어에서 가져온 headerHeight를 직접 사용
- 디바운스용 timeoutRef는 비동기 클린업을 필요해 꼭 필요한 ref라 삭제하지 않음

```tsx
const ref = useRef<HTMLDivElement>(null);
const headerHeight = useHeaderHeightStore((s) => s.headerHeight);
const setHeaderHeight = useHeaderHeightStore((s) => s.setHeaderHeight);
const timeoutRef = useRef<ReturnType<typeof setTimeout> | null>(null);

/**
 * headerHeight 업데이트
 ** NOTE: 이전 값과 비교하여 실제로 변경되었을 때만 실행
 */
const updateHeaderHeight = useEffectEvent((observingHeight: number) => {
  if (Math.abs(headerHeight - observingHeight) >= 1) {
    setHeaderHeight(observingHeight || 0);
  }
});

/** headerHeight 변경시 resizeObserver에서 실행시킬 콜백 함수 */
const onResize = useEffectEvent((entries: ResizeObserverEntry[]) => {
  // 이전 타이머 취소
  if (timeoutRef.current) clearTimeout(timeoutRef.current);

  // 디바운스를 통해 빈번한 업데이트 방지
  timeoutRef.current = setTimeout(() => {
    const observingHeight = entries[0].contentRect.height;
    updateHeaderHeight(observingHeight);
  }, 16); // ~60fps에 맞춘 디바운스
});

/** 초기 headerHeight 설정 */
const initHeight = useEffectEvent(() => {
  if (!ref.current) return;
  const initialHeight = Math.round(ref.current.offsetHeight);

  if (initialHeight !== headerHeight) {
    setHeaderHeight(initialHeight);
  }
});

useEffect(() => {
  if (!ref.current) return;

  const resizeObserver = new ResizeObserver(onResize); // [생성]
  resizeObserver.observe(ref.current); // [구독]
  initHeight(); // [초기화]

  // [정리]
  // eslint-disable-next-line consistent-return
  return () => {
    resizeObserver.disconnect();
    if (timeoutRef.current) clearTimeout(timeoutRef.current);
  };
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, []);
```
