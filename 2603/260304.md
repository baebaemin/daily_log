# Today I Learned

### [TIL-Words]

- `polyfill`
  - 브라우저가 지원하지 않는 API 또는 기능을 구현.
  - e.g., Promise 객체는 IE11에서 지원하지 않으므로 `polyfill`을 제공해야 한다.
- `ponyfill`
  - polyfill과 유사하지만, 전역 스코프에 영향을 주지 않는다.
  - e.g., promise-polyfill 패키지를 사용할 경우, 변수에 담아서 import한다 : `import Promise from 'promise-polyfill'`
  - 이렇게 하면 window.Promise는 여전히 undefined이므로 전역 스코프를 오염시키지 않는다.

### [git] 작업 중인 브랜치에서 다른 브랜치의 파일이나 일부 폴더만 가져오고 싶을 때

```zsh
git checkout {가져올브랜치명} -- {폴더 or 파일}
git checkout feature/goodbuilder-api -- .cursor
```

- 가져오면 자동으로 staged 된다.
- 가져올 때 복사해서 가져오므로 각 파일은 독립적임
