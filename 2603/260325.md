## Today I Learned

### [TIL-Words]

- SSH(Secure SHell): 네트워크를 통해 원격 서버에 안전하게 접속(터미널 명령어 조작)하는 방법.

  ```bash
  // 아래의 IP 주소의 서버에 접속 후 명령어 입력시 서버에서 실행됨
  ssh user@123.456.789.0
  ```

- 홈 서버(home server): 가정 내 PC나 미니 PC를 24시간 구동하여 NAS(데이터 저장), 웹 서비스, 스마트홈 허브 등으로 활용하는 개인 서버
  - 매일 IP가 바뀌므로 접속할 주소가 계속 달라진다 -> DDNS로 해결

- DDNS(Dynamic DNS): 유동 IP(가정, 소규모 사무실 등)에서 공유기나 서버의 IP 주소가 변경될 때, 등록된 도메인 이름을 자동으로 실시간 업데이트하여 항상 같은 주소로 접속할 수 있게 해주는 서비스
